# Makerspace Access

Minimal repository skeleton for a makerspace access system.

Structure:

- `api/` - FastAPI backend service
- `web/` - web frontend
- `ESP32/` - firmware and examples for ESP32-based door controllers
- `infra/` - infrastructure-as-code, deployment config
- `docs/` - design docs, runbooks, and specs

This repository intentionally contains only scaffolding. No business logic is implemented yet.

How to use
----------

Each top-level folder contains a minimal README with next steps. Add services and implementation into the appropriate directories.
